MVC (Model – View – Controller) Architecture

View

User Interface Layer

  • Displays data
  • HTML / JSP / Templates
  • Collects user input
➡️

Controller

Application Logic Layer

  • Handles requests
  • Calls Model methods
  • Selects appropriate View
➡️

Model

Data & Business Layer

  • Database operations
  • Business rules
  • Validations

MVC separates the application into three layers for easy development, maintenance, and testing.
