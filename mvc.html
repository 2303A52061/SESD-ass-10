MVC (Model - View - Controller) Architecture

View

User Interface Layer

  • Displays data to user
  • Receives input from user
  • HTML / JSP / Vue / React
  • No business logic
➡️

Controller

Application Logic Layer

  • Handles user requests
  • Processes input
  • Communicates with Model
  • Chooses which View to show
➡️

Model

Data & Business Logic

  • Database operations
  • Business rules
  • Data validation
  • CRUD operations
Data Flow: View ➜ Controller ➜ Model ➜ Controller ➜ View
